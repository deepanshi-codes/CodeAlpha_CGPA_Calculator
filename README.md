# CGPA Calculator (C++)

This project calculates the CGPA of a student based on course grades and credit hours.

## Features
- Takes number of courses as input
- Accepts grades and credit hours
- Calculates total credits
- Computes final CGPA

## Technologies Used
- C++
- VS Code

## How to Run

1. Compile the program:
 ```g++ cgpa.cpp -o cgpa```

3. Run the executable:
```./cgpa```
